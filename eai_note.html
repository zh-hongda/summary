                  以将leader机械臂设置绑定到ttyDXL_leader_right为例:

                  1. 插入leader机械臂到电脑

                  2. 确定设备名，通过检查插入机械臂前后/dev目录的变化. 设备名类似于：/dev/ttyUSB0

                    xxxxxxxxxx
                    2
                    1
                     # 使用自动化脚本检查设备名
                    2
                     python ./scripts/find_motors_bus_port.py

                  3. 确定设备序列号, 序列号类似于：FT88YWBJ

                    xxxxxxxxxx
                    1
                    1
                    udevadm info --name=/dev/ttyUSB0 --attribute-walk | grep ATTRS{serial} | head -n 1 | cut -d '"' -f2

                  4. 为设备添加符号链接, 将下面这行文本中"<SERIAL NUMBER>"替换为上面查询到的设备序列号，然后将修改后的文本添加到/etc/udev/rules.d/99-fixed-interbotix-udev.rules中，如果没有该文件新建一个即可

                    xxxxxxxxxx
                    1
                    1
                    SUBSYSTEM=="tty", ATTRS{serial}=="<SERIAL NUMBER>", ENV{ID_MM_DEVICE_IGNORE}="1", ATTR{device/latency_timer}="1", SYMLINK+="ttyDXL_master_right"

                  5. 更新规则

                    xxxxxxxxxx
                    1
                    1
                    sudo udevadm control --reload && sudo udevadm trigger

                  def ModifiedIKinSpace(Slist, M, T, thetalist0, eomg, ev, maxiterations=40):
                  47
                      """
                  48
                      ModifiedIKinSpace - Inverse Kinematics in the Space Frame
                  49
                      this exposed the max_iterations parameter to the user
                  50
                  

                  51
                      # original source:
                  52
                      ModernRobotics/packages/Python/modern_robotics/core.py at master · NxRLab/ModernRobotics
                  53
                      """
                  54
                      thetalist = np.array(thetalist0).copy()
                  55
                      i = 0
                  56
                      Tsb = FKinSpace(M,Slist, thetalist)
                  57
                      Vs = np.dot(Adjoint(Tsb), \
                  58
                                  se3ToVec(MatrixLog6(np.dot(TransInv(Tsb), T))))
                  59
                      err = np.linalg.norm([Vs[0], Vs[1], Vs[2]]) > eomg \
                  60
                            or np.linalg.norm([Vs[3], Vs[4], Vs[5]]) > ev
                  61
                      while err and i < maxiterations:
                  62
                          thetalist = thetalist \
                  63
                                      + np.dot(np.linalg.pinv(JacobianSpace(Slist, \
                  64
                                                                            thetalist)), Vs)
                  65
                          i = i + 1
                  66
                          Tsb = FKinSpace(M, Slist, thetalist)
                  67
                          Vs = np.dot(Adjoint(Tsb), \
                  68
                                      se3ToVec(MatrixLog6(np.dot(TransInv(Tsb), T))))
                  69
                          err = np.linalg.norm([Vs[0], Vs[1], Vs[2]]) > eomg \
                  70
                                or np.linalg.norm([Vs[3], Vs[4], Vs[5]]) > ev
                  71
                      if err:
